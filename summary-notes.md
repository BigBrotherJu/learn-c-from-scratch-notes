# 全书总结

## gcc 使用

- `a.out`（Assembler Output）
- `-o` 参数
- `-Wall` 选项

## 类型（Type）

- 字符（Character）类型

  `char`

- 整数（Integer）类型

  `int`

- 浮点数（Floating Point）类型

  - 单精度浮点数类型

    `float`

  - 双精度浮点数类型

    `double`

  - 精度更高的浮点数类型

    `long double`

- 函数类型（Function Type）

## 常量（Constant）

- 字符常量
- 整数常量
- 浮点数常量

  3.14 是 `double` 类型的。

- 枚举常量
